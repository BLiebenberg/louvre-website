# Mobile First Rules

## Purpose
Make sure the site works beautifully on phones first.

## Rules
- Start every layout at mobile width.
- Stack buttons full-width where needed.
- Keep logo readable but not oversized.
- Avoid tiny navigation text.
- Use one-column sections on mobile.
- Keep maps usable but not dominant.
- Crop images deliberately.
- Ensure no text overlaps images.

## Mobile CTAs
- Call button should be easy to reach.
- Booking buttons must be visible.
- Quote actions must be obvious.

## Testing
- Test iPhone narrow width.
- Test Android common width.
- Test long menu labels.
- Test image crops.
- Test contact links.
