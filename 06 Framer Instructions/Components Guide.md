# Components Guide

## Purpose
Define reusable components for the Louvre website.

## Components
### Header
Logo, navigation, and call or booking action.

### Booking Button Group
Book Guesthouse, Book Manor House, Ask for a Quote.

### Property Card
Image, eyebrow, property name, short description, CTA.

### Feature Section
Image or image pair plus focused copy and CTA.

### Map Block
Address, check-in details, directions button, large embedded map.

### Contact Panel
Phone, email, quote action, booking links.

### SEO Content Block
Short section with local keyword support and practical details.

## Rules
- Components must use paper background and charcoal text.
- Borders and dividers are 1px charcoal.
- Use consistent image ratios.
