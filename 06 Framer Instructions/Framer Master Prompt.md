# Framer Master Prompt

## Purpose
Guide Framer or AI-assisted build work so it matches Louvre's brand and website strategy.

## Prompt
Build a refined, mobile-first website for Louvre, a hospitality collection in Bothaville. The site must feel calm, established, practical, and elegant. Use a strict paper and charcoal palette, Calibri-style typography, thin 1px charcoal dividers, real photography, and clear booking actions.

## Site Structure
- Home.
- Stay.
- Gather.
- Explore.
- Work Well.
- About.
- Contact.
- Book Now.

## Brand Rules
- Louvre is one hospitality system with multiple properties.
- Do not make it feel like a Shopify store.
- Keep booking effortless.
- Use real images as proof.
- Avoid decorative gradients, colourful blocks, and generic travel styling.

## Primary CTAs
- Book Guesthouse.
- Book Manor House.
- Ask for a Quote.
- Call 084 083 4000.
