# Hero Images

## Purpose
Define strong first-screen images for key pages.

## Home Hero
Ideal: Louvre exterior or Gallery evening image with life, warmth, and sense of place.

## Stay Hero
Ideal: clean room or property image that immediately reads as accommodation.

## Gather Hero
Ideal: Gallery set with people or a refined function setup.

## Work Well Hero
Ideal: room, desk, breakfast, or parking image that supports business travel.

## Explore Hero
Ideal: Bothaville, NAMPO, or local context image.

## Rules
- Avoid dark images where details disappear.
- Avoid images that crop the important subject.
- Use real Louvre images, not generic stock.
- Test on mobile first.

## Next Actions
- Select one hero candidate per page.
- Export desktop and mobile crops.
