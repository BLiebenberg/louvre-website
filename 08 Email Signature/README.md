# Louvre Email Signature

## Direction

The signature is deliberately compact and functional:

- Ben's name and role lead.
- One Louvre logo.
- Phone, email, website, and address remain visible.
- One clear `BOOK DIRECT` action.
- No social-media rows, promotional slogans, or dark legal bar.
- Calibri, charcoal, white, and Louvre paper-clay only.
- Website Charcoal Tape: `#69655D`.

## Files

- `Louvre - Ben Liebenberg.htm` is the email-safe signature.
- `Preview.htm` shows the signature below a short sample email.
- `Louvre - Ben Liebenberg_files/louvre-logo-transparent.png` preserves the original `1575 x 483` logo canvas and design exactly, with Charcoal Tape `#69655D` artwork on a transparent background.
- The signature loads this asset from its public GitHub HTTPS URL so it remains visible in New Outlook and received emails.

## Installation

### Classic Outlook for Windows

1. Close Outlook.
2. Open `%APPDATA%\Microsoft\Signatures`.
3. Copy `Louvre - Ben Liebenberg.htm` and the complete `Louvre - Ben Liebenberg_files` folder into it.
4. Open Outlook and select the signature under **File > Options > Mail > Signatures**.

### New Outlook or Outlook Web

1. Open `Preview.htm` in a browser.
2. Select only the signature.
3. Copy and paste it into **Settings > Accounts > Signatures**.
4. Send a test email to Gmail and Outlook before making it the default.
