# Performance Checklist

## Purpose
Keep the website fast, stable, and mobile friendly.

## Image Performance
- Compress images.
- Export correct dimensions.
- Use modern formats where possible.
- Lazy load below-the-fold images.
- Avoid oversized hero images.

## Layout Performance
- Prevent layout shift.
- Set image dimensions or ratios.
- Keep fonts simple.
- Avoid heavy animation.

## Technical Checks
- PageSpeed mobile.
- PageSpeed desktop.
- Core Web Vitals.
- Broken links.
- 404 pages.
- Redirects.

## Targets
- Fast first load on mobile.
- No obvious layout shift.
- Images sharp but not heavy.

## Next Actions
- Run PageSpeed before launch.
- Optimise worst images first.
