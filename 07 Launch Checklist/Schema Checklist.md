# Schema Checklist

## Purpose
Add structured data that helps search engines understand Louvre.

## Schema Types
- LodgingBusiness.
- Hotel.
- LocalBusiness.
- EventVenue if appropriate for The Gallery.
- FAQPage for common booking questions.
- BreadcrumbList.

## Required Details
- Business name.
- Address.
- Phone.
- Website.
- Logo.
- Image.
- Opening or contact hours.
- Geo coordinates if available.
- SameAs social or profile links.

## Page-Specific Schema
- Home: LocalBusiness or Organization.
- Property pages: LodgingBusiness or Hotel.
- Gallery page: EventVenue or LocalBusiness.
- FAQ blocks: FAQPage.
