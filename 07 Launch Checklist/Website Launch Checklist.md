# Website Launch Checklist

## Purpose
Track what must be done before the new Louvre website goes live.

## Content
- Final page copy approved.
- Property details checked.
- Contact details checked.
- Addresses consistent.
- Check-in and check-out times consistent.
- Booking links tested.
- Gallery facts confirmed.

## Design
- Mobile checked.
- Desktop checked.
- Images cropped.
- Logo crisp.
- Paper and charcoal palette consistent.

## Technical
- Domain connected.
- SSL active.
- Forms tested.
- Analytics installed.
- Search Console connected.
- Meta Pixel installed if used.
- Schema added.

## Final Review
- Test booking flow.
- Test phone links.
- Test email links.
- Test map links.
